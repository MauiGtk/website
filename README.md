# MauiGtk website

The MauiGtk Community website/blog for documentations and promoting.

## Usage

### Requires

- Git
- Go
- Hugo extended
- npm

```sh
# Install tailwindcss
sudo npm install -g tailwindcss
```

### Run

```sh
npm i # Once
npm run start
```

### Build

### Update theme

```sh
git submodule update --remote --merge
```
