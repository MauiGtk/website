# MauiGtk website

The MauiGtk Community website/blog for documentations and promoting. Build with the Hugo static site generator.

## Usage

### Requires

- Go
- Hugo Extended
- npm

### Build

```sh
# Update modules
# hugo mod tidy

# node dependencies
# hugo mod npm pack

# Install dependencies
npm install

# Run
hugo server -w
```
